## Исполняемый код для статьи "Опорные условия для выпуклых комапактных множеств. Методы проекции градиента и условного градиента"

Структура репозитория:
- Папка `images` содержит иллюстрации из статьи.
- Папка `html` содержит интеракивную версию рис. 3 в формате HTML-файла.
- В файле `experiments_ru.ipynb` описаны эксперименты из статьи.
- В файле `gradient.py` содержится реализация градиентных методов оптимизации.
- В файле `sets.py` содержится реализация выпуклых множеств, используемых в экспериментах. Множества описываются при помощии опорных функций.
- В файле `lattice.py` содержится реализация адаптивной сетки нормалей, 
используемой для отрисовки выпуклых множеств.
- В файле `drawing.py` содержатся процедуры для отрисовки выпуклых множеств.

Repository structure:
